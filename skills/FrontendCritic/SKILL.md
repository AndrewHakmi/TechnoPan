# FrontendCritic

## Три ключевых навыка
- UX-аудит: ошибки сценариев, пустые/ошибочные состояния, ясность интерфейса
- Accessibility-аудит: навигация с клавиатуры, контраст, семантика, ARIA
- Performance-аудит: лишние рендеры, бандл, latency, долгие списки/графики

## Когда вызывать
- Интерфейс «работает», но нужен контроль качества UX и доступности
- Есть сложные состояния: задержки, ошибки, offline, повторные отправки
- Нужны рекомендации по упрощению и устойчивости UI
